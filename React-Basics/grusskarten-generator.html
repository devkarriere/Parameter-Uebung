Übung: Der Grußkarten-Generator

React-Komponenten, Props & State in der Praxis

Geführte Aufgabe: Der Grußkarten-Generator

In dieser Übung baust du Schritt für Schritt einen Grußkarten-Generator. Der Nutzer kann einen Namen eingeben, eine Nachricht schreiben und eine Stimmung wählen – die Karte aktualisiert sich sofort!

Vorbereitung: Öffne das Terminal und erstelle ein neues React-Projekt mit Create React App: 
npx create-react-app grusskarte
cd grusskarte
npm start

Der Befehl npx create-react-app grusskarte erstellt einen neuen Ordner grusskarte mit allem, was du brauchst. npm start startet einen Entwicklungsserver – dein Browser öffnet sich automatisch auf localhost:3000.

Deinen Code schreibst du in die Datei src/App.js. Ersetze den bestehenden Inhalt komplett durch deinen eigenen Code. Die Seite aktualisiert sich bei jeder Änderung automatisch!

Öffne zusätzlich die Konsole im Browser (F12 → Console), um Fehler zu sehen.

1

Die Grußkarte als Komponente

Erstelle eine Komponente GreetingCard, die einen einfachen Gruß anzeigt. Die Komponente gibt JSX zurück – das ist der sichtbare Teil.

Lösung anzeigen 
// src/App.js
function GreetingCard() {
  return (
    <div>
      <p style={{fontSize: "3rem"}}>👋</p>
      <h2>Hallo!</h2>
      <p>Willkommen bei React!</p>
    </div>
  );
}
Beachte: className statt class! Die Zeile export default GreetingCard sagt React: „Das ist die Komponente, die diese Datei nach außen bereitstellt.“ Create React App zeigt automatisch die exportierte Komponente aus src/App.js an. Teste den Code im Browser – du solltest den Gruß sehen.
2

Props für den Namen

Erweitere die Komponente um einen Props-Parameter, damit der Name personalisiert werden kann. Die Komponente soll den Namen anzeigen, der von außen übergeben wird.

Lösung anzeigen 
// src/App.js
function GreetingCard(props) {
  return (
    <div>
      <p style={{fontSize: "3rem"}}>👋</p>
      <h2>Hallo, {props.name}!</h2>
      <p>Schön, dass du da bist!</p>
    </div>
  );
}

// App-Komponente ruft GreetingCard mit einem Prop auf:
function App() {
  return <GreetingCard name="Anna" />;
}

export default App;
name="Anna" Attribut → props.name "Anna" → {props.name} im JSX

Der Wert von name fließt als props.name in die Komponente.

Tipp: Der Wert in geschweiften Klammern {props.name} wird durch den tatsächlichen Wert ersetzt. Probiere verschiedene Namen aus!
3

Mehr Props – Nachricht und Stimmung

Füge zwei weitere Props hinzu: message für eine persönliche Nachricht und mood für die Stimmung (z.B. "happy", "love", "cool"). Die Stimmung soll das Emoji und die Farbe der Karte ändern.

Lösung anzeigen 
// src/App.js
function GreetingCard(props) {
  let emoji;
  if (props.mood === "happy") {
    emoji = "😊";
  } else if (props.mood === "love") {
    emoji = "❤️";
  } else if (props.mood === "cool") {
    emoji = "😎";
  } else {
    emoji = "👋";
  }

  return (
    <div className={"greeting-card " + props.mood}>
      <div className="emoji">{emoji}</div>
      <h2>Hallo, {props.name}!</h2>
      <p className="message">{props.message}</p>
    </div>
  );
}

function App() {
  return (
    <GreetingCard
      name="Max"
      message="Alles Gute zum Geburtstag!"
      mood="happy"
    />
  );
}

export default App;
name="Max" Prop 1 message="..." Prop 2 mood="happy" Prop 3

Alle drei Props fließen in die Komponente und steuern Name, Nachricht und Aussehen.

Beachte: Hier siehst du bedingte Anzeige in Aktion: Je nach mood wird ein anderes Emoji angezeigt und eine andere CSS-Klasse gesetzt. Die Klasse ändert die Rahmenfarbe und den Hintergrund der Karte.
4

useState für das Formular

Erstelle eine neue Komponente CardCreator, die ein Formular anzeigt. Verwende useState, damit React sich merkt, was der Nutzer eingibt. Du brauchst drei State-Variablen: name, message und mood.

Die CardCreator-Komponente soll die GreetingCard verwenden und ihr die State-Werte als Props übergeben.

Lösung anzeigen 
// src/App.js
import { useState } from "react";

function CardCreator() {
  const [name, setName] = useState("");
  const [message, setMessage] = useState("");
  const [mood, setMood] = useState("happy");

  function handleNameChange(event) {
    setName(event.target.value);
  }

  function handleMessageChange(event) {
    setMessage(event.target.value);
  }

  function handleMoodChange(newMood) {
    setMood(newMood);
  }

  return (
    <div>
      <div className="form-group">
        <label>Name:</label>
        <input
          type="text"
          value={name}
          onChange={handleNameChange}
          placeholder="Für wen ist die Karte?"
        />
      </div>

      <div className="form-group">
        <label>Nachricht:</label>
        <input
          type="text"
          value={message}
          onChange={handleMessageChange}
          placeholder="Deine persönliche Nachricht..."
        />
      </div>

      <div className="form-group">
        <label>Stimmung:</label>
        <div className="mood-buttons">
          <button
            className={"mood-btn" + (mood === "happy" ? " active" : "")}
            onClick={function() { handleMoodChange("happy"); }}
          >😊 Fröhlich</button>
          <button
            className={"mood-btn" + (mood === "love" ? " active" : "")}
            onClick={function() { handleMoodChange("love"); }}
          >❤️ Liebe</button>
          <button
            className={"mood-btn" + (mood === "cool" ? " active" : "")}
            onClick={function() { handleMoodChange("cool"); }}
          >😎 Cool</button>
        </div>
      </div>

      <GreetingCard name={name} message={message} mood={mood} />
    </div>
  );
}
Neu: Die Zeile import { useState } from "react" holt die useState-Funktion direkt aus React. In Create React App nutzt du immer import statt React.useState.
onChange Eingabe → setName("Anna") State ändern → name="Anna" als Prop

Jede Eingabe löst onChange aus, der State wird aktualisiert, und die Karte rendert sich neu.

Wichtig: Hier passieren zwei wichtige Dinge: onChange ist ein Callback, der bei jeder Tastatureingabe ausgelöst wird. setName ändert den State, und React rendert automatisch neu – die Grußkarte aktualisiert sich sofort!
5

Alles zusammenfügen mit der App-Komponente

Erstelle eine App-Komponente, die alles zusammenfasst. Die App rendert den CardCreator, und der CardCreator rendert die GreetingCard. So entsteht ein Komponenten-Baum.

Lösung anzeigen 
// src/App.js – der vollständige Code:
import { useState } from "react";

function GreetingCard(props) {
  // ... (wie in Schritt 3)
}

function CardCreator() {
  // ... (wie in Schritt 4)
}

function App() {
  return (
    <div>
      <h1>Grußkarten-Generator</h1>
      <CardCreator />
    </div>
  );
}

export default App;

Komponenten-Baum:

App
CardCreator (State: name, message, mood)
GreetingCard (Props: name, message, mood)
Beachte: Die Daten fließen von oben nach unten: CardCreator hat den State und gibt ihn als Props an GreetingCard weiter. Das nennt man Datenfluss von oben nach unten (Top-Down Data Flow).

Bonus: Schriftgröße anpassen

Füge einen weiteren State fontSize hinzu (z.B. "klein", "mittel", "groß") und erstelle drei Buttons, mit denen der Nutzer die Schriftgröße der Nachricht ändern kann. Übergib fontSize als Prop an die GreetingCard.

Lösung anzeigen 
// In CardCreator hinzufügen:
const [fontSize, setFontSize] = useState("mittel");

function handleFontSizeChange(newSize) {
  setFontSize(newSize);
}

// Neue Buttons im Formular:
<div className="form-group">
  <label>Schriftgröße:</label>
  <div className="mood-buttons">
    <button
      className={"mood-btn" + (fontSize === "klein" ? " active" : "")}
      onClick={function() { handleFontSizeChange("klein"); }}
    >Klein</button>
    <button
      className={"mood-btn" + (fontSize === "mittel" ? " active" : "")}
      onClick={function() { handleFontSizeChange("mittel"); }}
    >Mittel</button>
    <button
      className={"mood-btn" + (fontSize === "gross" ? " active" : "")}
      onClick={function() { handleFontSizeChange("gross"); }}
    >Groß</button>
  </div>
</div>

// In GreetingCard die Prop nutzen:
let textSize;
if (props.fontSize === "klein") {
  textSize = "0.9rem";
} else if (props.fontSize === "gross") {
  textSize = "1.4rem";
} else {
  textSize = "1.1rem";
}

<p className="message" style={{fontSize: textSize}}>
  {props.message}
</p>
