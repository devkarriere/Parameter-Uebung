Komplexe Aufgabe: Das Mini-Quiz

In dieser Übung baust du eine Quiz-App mit mehreren Fragen, einem Punkte-System, einem Fortschrittsbalken und einem Ergebnis-Bildschirm. Hier kommen alle Konzepte zusammen!

Vorbereitung: Erstelle ein neues React-Projekt wie in den vorherigen Übungen: 
npx create-react-app quiz
cd quiz
npm start
Deinen Code schreibst du wieder in src/App.js.

2

Frage-Komponente (QuestionCard)

Erstelle eine Komponente QuestionCard mit diesen Props: question (der Fragetext), answers (die Antworten) und onAnswer (ein Callback, der beim Klick auf eine Antwort aufgerufen wird). Zeige die Frage und vier Antwort-Buttons an.

Lösung anzeigen 
function QuestionCard(props) {
  function handleAnswer(index) {
    props.onAnswer(index);
  }

  return (
    <div className="question-card">
      <p className="question-text">{props.question}</p>
      {props.answers.map(function(answer, index) {
        return (
          <button
            key={index}
            className="answer-btn"
            onClick={function() { handleAnswer(index); }}
          >
            {answer}
          </button>
        );
      })}
    </div>
  );
}
3

Antwort-Buttons mit Feedback

Erweitere QuestionCard um zwei weitere Props: selectedAnswer (der Index der gewählten Antwort oder null) und correctAnswer (der Index der richtigen Antwort). Nachdem geantwortet wurde, sollen die Buttons farbig werden: grün für richtig, rot für falsch. Zeige außerdem eine Feedback-Nachricht an.

Lösung anzeigen 
function QuestionCard(props) {
  let hasAnswered = props.selectedAnswer !== null;

  function getButtonClass(index) {
    if (!hasAnswered) return "answer-btn";

    if (index === props.correctAnswer) {
      return "answer-btn correct";
    }
    if (index === props.selectedAnswer) {
      return "answer-btn wrong";
    }
    return "answer-btn disabled";
  }

  function handleAnswer(index) {
    if (!hasAnswered) {
      props.onAnswer(index);
    }
  }

  return (
    <div className={"question-card" + (hasAnswered
      ? (props.selectedAnswer === props.correctAnswer ? " correct" : " wrong")
      : "")}>
      <p className="question-text">{props.question}</p>
      {props.answers.map(function(answer, index) {
        return (
          <button
            key={index}
            className={getButtonClass(index)}
            onClick={function() { handleAnswer(index); }}
          >
            {answer}
          </button>
        );
      })}
      {hasAnswered ? (
        <div className={"feedback " + (props.selectedAnswer === props.correctAnswer ? "correct" : "wrong")}>
          {props.selectedAnswer === props.correctAnswer
            ? "Richtig! 🎉"
            : "Falsch! Die richtige Antwort war: " + props.answers[props.correctAnswer]}
        </div>
      ) : null}
    </div>
  );
}
Beachte: Die Funktion getButtonClass entscheidet für jeden Button, welche CSS-Klasse er bekommt. Das ist eine gute Methode, um komplexe bedingte Logik aus dem JSX herauszuhalten.

Fortschrittsbalken (ProgressBar)

Erstelle eine Komponente ProgressBar mit den Props current und total. Berechne die Breite in Prozent und setze sie als style.

Lösung anzeigen 
function ProgressBar(props) {
  let percent = (props.current / props.total) * 100;

  return (
    <div className="progress-bar">
      <div
        className="progress-fill"
        style={{width: percent + "%"}}
      ></div>
    </div>
  );
}
Tipp: In JSX schreibst du Inline-Styles als Objekt: style={{width: percent + "%"}}. Beachte die doppelten geschweiften Klammern – die äußeren sagen „hier kommt JavaScript“, die inneren definieren das Objekt.
6

Quiz-App zusammenbauen

Erstelle die Hauptkomponente QuizApp. Du brauchst mehrere useState-Aufrufe:

  • currentQuestion – Index der aktuellen Frage (Start: 0)
  • score – Punktestand (Start: 0)
  • selectedAnswer – Gewählte Antwort (Start: null)
  • isFinished – Quiz beendet? (Start: false)

Schreibe eine Funktion handleAnswer, die prüft ob die Antwort richtig ist und den Score aktualisiert. Und eine Funktion handleNext, die zur nächsten Frage weitergeht oder das Quiz beendet.

Lösung anzeigen 
import { useState } from "react";

function QuizApp() {
  const [currentQuestion, setCurrentQuestion] = useState(0);
  const [score, setScore] = useState(0);
  const [selectedAnswer, setSelectedAnswer] = useState(null);
  const [isFinished, setIsFinished] = useState(false);

  let question = questions[currentQuestion];

  function handleAnswer(answerIndex) {
    setSelectedAnswer(answerIndex);
    if (answerIndex === question.correct) {
      setScore(score + 1);
    }
  }

  function handleNext() {
    if (currentQuestion + 1 < questions.length) {
      setCurrentQuestion(currentQuestion + 1);
      setSelectedAnswer(null);
    } else {
      setIsFinished(true);
    }
  }

  function handleRestart() {
    setCurrentQuestion(0);
    setScore(0);
    setSelectedAnswer(null);
    setIsFinished(false);
  }

  if (isFinished) {
    return (
      <div className="quiz-container">
        <ResultScreen
          score={score}
          total={questions.length}
          onRestart={handleRestart}
        />
      </div>
    );
  }

  return (
    <div className="quiz-container">
      <ProgressBar
        current={currentQuestion + 1}
        total={questions.length}
      />
      <p className="question-number">
        Frage {currentQuestion + 1} von {questions.length}
      </p>
      <ScoreDisplay score={score} total={questions.length} />
      <QuestionCard
        question={question.question}
        answers={question.answers}
        correctAnswer={question.correct}
        selectedAnswer={selectedAnswer}
        onAnswer={handleAnswer}
      />
      {selectedAnswer !== null ? (
        <button className="next-btn" onClick={handleNext}>
          {currentQuestion + 1 < questions.length
            ? "Nächste Frage"
            : "Ergebnis anzeigen"}
        </button>
      ) : null}
    </div>
  );
}

export default QuizApp;

Komponenten-Baum:

QuizApp (State: currentQuestion, score, selectedAnswer, isFinished)
ProgressBar (Props: current, total)
ScoreDisplay (Props: score, total)
QuestionCard (Props: question, answers, correctAnswer, selectedAnswer, onAnswer)
Klick! → props.onAnswer(index) → handleAnswer(index) → setScore, setSelectedAnswer
ResultScreen (Props: score, total, onRestart) – wenn isFinished
7

Ergebnis-Bildschirm (ResultScreen)

Erstelle eine Komponente ResultScreen mit den Props score, total und onRestart (Callback für den Neustart-Button). Zeige ein Emoji und eine Nachricht basierend auf dem Ergebnis: 80%+ = Feier-Emoji, 50%+ = Daumen hoch, darunter = Ermutigend.

Lösung anzeigen 
function ResultScreen(props) {
  let percent = (props.score / props.total) * 100;
  let emoji;
  let message;

  if (percent >= 80) {
    emoji = "🎉";
    message = "Ausgezeichnet! Du bist ein Profi!";
  } else if (percent >= 50) {
    emoji = "👍";
    message = "Gut gemacht! Weiter so!";
  } else {
    emoji = "💪";
    message = "Übe weiter, du schaffst das!";
  }

  return (
    <div className="result-screen">
      <div className="result-emoji">{emoji}</div>
      <p className="result-text">{message}</p>
      <p className="result-score">
        {props.score} von {props.total} Fragen richtig ({Math.round(percent)}%)
      </p>
      <button className="restart-btn" onClick={props.onRestart}>
        Quiz neu starten
      </button>
    </div>
  );
}

Bonus: Zufällige Fragen-Reihenfolge

Mische die Fragen bei jedem Quiz-Start zufällig durch. Erstelle dafür eine Funktion shuffleArray, die ein neues, gemischtes Array zurückgibt. Speichere die gemischten Fragen als State.

Lösung anzeigen 
function shuffleArray(array) {
  let shuffled = [...array];
  for (let i = shuffled.length - 1; i > 0; i--) {
    let j = Math.floor(Math.random() * (i + 1));
    let temp = shuffled[i];
    shuffled[i] = shuffled[j];
    shuffled[j] = temp;
  }
  return shuffled;
}

// In QuizApp:
const [shuffledQuestions, setShuffledQuestions] =
  useState(shuffleArray(questions));

// Nutze shuffledQuestions statt questions:
let question = shuffledQuestions[currentQuestion];

// Bei handleRestart auch neu mischen:
function handleRestart() {
  setShuffledQuestions(shuffleArray(questions));
  setCurrentQuestion(0);
  setScore(0);
  setSelectedAnswer(null);
  setIsFinished(false);
}
