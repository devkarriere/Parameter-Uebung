Bewertungen mit Emojis – Props, bedingte Anzeige & Callbacks

Aufbauende Aufgabe: Die Emoji-Bewertung

In dieser Übung baust du ein Bewertungssystem mit Emojis. Nutzer können eine Bewertung von 1 bis 5 abgeben, vergangene Bewertungen sehen und den Durchschnitt berechnen.

Vorbereitung: Erstelle ein neues React-Projekt wie in Übung 1: 
npx create-react-app bewertung
cd bewertung
npm start
Deinen Code schreibst du wieder in src/App.js.
1

Emoji-Anzeige als Komponente

Erstelle eine Komponente EmojiDisplay, die eine Prop rating (Zahl von 1 bis 5) bekommt und das passende Emoji anzeigt. Nutze if/else für die Zuordnung.

Emoji-Zuordnung: 1 = 😡, 2 = 😕, 3 = 😐, 4 = 🙂, 5 = 😍

Lösung anzeigen 
function EmojiDisplay(props) {
  let emoji;
  if (props.rating === 1) {
    emoji = "😡";
  } else if (props.rating === 2) {
    emoji = "😕";
  } else if (props.rating === 3) {
    emoji = "😐";
  } else if (props.rating === 4) {
    emoji = "🙂";
  } else if (props.rating === 5) {
    emoji = "😍";
  } else {
    emoji = "❓";
  }

  return <span style={{fontSize: "3rem"}}>{emoji}</span>;
}
Tipp: Teste die Komponente mit verschiedenen Werten: <EmojiDisplay rating={3} />. Denk daran, dass Zahlen in geschweiften Klammern stehen!
2

Bewertungslabel mit bedingter Anzeige

Erstelle eine Komponente RatingLabel, die eine Prop rating bekommt und einen Text mit passender Farbe anzeigt: 1–2 = "Schlecht" (rot), 3 = "Geht so" (orange), 4–5 = "Super!" (grün).

Lösung anzeigen 
function RatingLabel(props) {
  let text;
  let colorClass;

  if (props.rating <= 2) {
    text = "Schlecht";
    colorClass = "bad";
  } else if (props.rating === 3) {
    text = "Geht so";
    colorClass = "ok";
  } else {
    text = "Super!";
    colorClass = "good";
  }

  return <p className={"rating-label " + colorClass}>{text}</p>;
}
Beachte: Die CSS-Klasse wird dynamisch zusammengesetzt: {"rating-label " + colorClass}. So ändert sich die Farbe je nach Bewertung.
3

Klickbare Emoji-Buttons

Erstelle eine Komponente EmojiSelector mit zwei Props: onSelect (eine Callback-Funktion) und selected (die aktuelle Auswahl). Zeige 5 Emoji-Buttons. Beim Klick wird props.onSelect(wert) aufgerufen. Der ausgewählte Button bekommt die Klasse selected.

Lösung anzeigen 
function EmojiSelector(props) {
  const emojis = ["😡", "😕", "😐", "🙂", "😍"];

  function handleClick(value) {
    props.onSelect(value);
  }

  return (
    <div className="emoji-row">
      {emojis.map(function(emoji, index) {
        let value = index + 1;
        return (
          <button
            key={value}
            className={"emoji-btn" + (props.selected === value ? " selected" : "")}
            onClick={function() { handleClick(value); }}
          >
            {emoji}
          </button>
        );
      })}
    </div>
  );
}
onClick Klick auf 🙂 → props.onSelect(4) Callback → setRating(4) im Eltern

Der Klick löst den Callback aus, der den State im Eltern-Element ändert.

4

useState für die Bewertung

Erstelle eine Komponente RatingWidget, die alles zusammenbringt. Verwende useState für die aktuelle Bewertung (Startwert: 0). Übergib setRating als Callback an EmojiSelector und die rating-Variable als Prop an EmojiDisplay und RatingLabel.

Lösung anzeigen 
import { useState } from "react";

function RatingWidget() {
  const [rating, setRating] = useState(0);

  return (
    <div className="rating-display">
      <h3>Wie findest du es?</h3>
      <EmojiSelector selected={rating} onSelect={setRating} />
      {rating > 0 ? (
        <div>
          <EmojiDisplay rating={rating} />
          <RatingLabel rating={rating} />
        </div>
      ) : (
        <p style={{color: "#64748b"}}>Wähle eine Bewertung...</p>
      )}
    </div>
  );
}

export default RatingWidget;
Neu hier: Der Ausdruck {rating > 0 ? (...) : (...)} zeigt die Emoji-Anzeige nur, wenn eine Bewertung ausgewählt wurde. Sonst erscheint ein Platzhalter-Text.
5

Bewertungsliste mit mehreren Bewertungen

Erweitere RatingWidget um einen zweiten State reviews (ein leeres Array als Startwert). Füge einen Button "Bewertung abgeben" hinzu. Beim Klick wird die aktuelle Bewertung zum Array hinzugefügt. Zeige alle abgegebenen Bewertungen als Liste an.

Lösung anzeigen 
import { useState } from "react";

function RatingWidget() {
  const [rating, setRating] = useState(0);
  const [reviews, setReviews] = useState([]);

  function handleSubmit() {
    if (rating === 0) return;
    setReviews([...reviews, rating]);
    setRating(0);
  }

  return (
    <div>
      <div className="rating-display">
        <h3>Wie findest du es?</h3>
        <EmojiSelector selected={rating} onSelect={setRating} />
        {rating > 0 ? (
          <div>
            <EmojiDisplay rating={rating} />
            <RatingLabel rating={rating} />
            <button onClick={handleSubmit}>Bewertung abgeben</button>
          </div>
        ) : (
          <p style={{color: "#64748b"}}>Wähle eine Bewertung...</p>
        )}
      </div>

      <h3 style={{color: "#94a3b8", marginTop: "1.5rem"}}>
        Bisherige Bewertungen ({reviews.length}):
      </h3>
      {reviews.length === 0 ? (
        <p className="no-reviews">Noch keine Bewertungen abgegeben.</p>
      ) : (
        reviews.map(function(rev, index) {
          return (
            <div className="review-card" key={index}>
              <span className="review-emoji">
                <EmojiDisplay rating={rev} />
              </span>
              <span className="review-text">
                Bewertung #{index + 1}: {rev} von 5
              </span>
            </div>
          );
        })
      )}
    </div>
  );
}
Wichtig: [...reviews, rating] erstellt ein neues Array mit allen bisherigen Bewertungen plus der neuen. In React darfst du ein Array nie direkt verändern (z.B. mit push), sondern musst immer ein neues Array erstellen!

Durchschnitt berechnen und anzeigen

Erstelle eine Komponente AverageDisplay, die eine Prop reviews (das Array der Bewertungen) bekommt. Berechne den Durchschnitt und zeige ihn an. Verwende die Komponente in RatingWidget.

Lösung anzeigen 
function AverageDisplay(props) {
  if (props.reviews.length === 0) {
    return null;
  }

  let total = 0;
  for (let i = 0; i < props.reviews.length; i++) {
    total = total + props.reviews[i];
  }
  let average = total / props.reviews.length;

  return (
    <div className="average-display">
      Durchschnitt: {average.toFixed(1)} von 5
      <span style={{marginLeft: "0.5rem"}}>
        <EmojiDisplay rating={Math.round(average)} />
      </span>
    </div>
  );
}

// In RatingWidget nach der Bewertungsliste einfügen:
<AverageDisplay reviews={reviews} />

Bonus: Kommentar zur Bewertung

Füge ein Eingabefeld für einen Kommentar hinzu. Speichere die Bewertungen als Objekte: {rating: 4, comment: "Sehr gut!"}. Zeige den Kommentar in der Bewertungsliste an.

Lösung anzeigen 
// Neuer State für den Kommentar:
const [comment, setComment] = useState("");

function handleCommentChange(event) {
  setComment(event.target.value);
}

// Eingabefeld im Formular:
<div className="form-group" style={{marginTop: "0.5rem"}}>
  <input
    type="text"
    value={comment}
    onChange={handleCommentChange}
    placeholder="Kommentar (optional)..."
  />
</div>

// handleSubmit anpassen:
function handleSubmit() {
  if (rating === 0) return;
  setReviews([...reviews, {rating: rating, comment: comment}]);
  setRating(0);
  setComment("");
}

// In der Bewertungsliste:
<span className="review-text">
  Bewertung #{index + 1}: {rev.rating} von 5
  {rev.comment ? " – " + rev.comment : ""}
</span>
