Parameter & Variablen in JavaScript

Verstehe den Unterschied – Schritt für Schritt

1. Was sind Variablen?

Eine Variable ist ein Behälter mit einem Namen, in dem du einen Wert speicherst. 

let name = "Anna";
let age = 25;

console.log(name); // "Anna"
console.log(age);  // 25
Merke: let erstellt eine Variable. Der Name links vom = ist frei wählbar.

2. Was sind Parameter?

Ein Parameter ist eine Platzhalter-Variable innerhalb einer Funktion. Er bekommt seinen Wert erst, wenn die Funktion aufgerufen wird. 

function greet(firstName) {
  console.log("Hallo, " + firstName + "!");
}

greet("Max");   // "Hallo, Max!"
greet("Lisa");  // "Hallo, Lisa!"
Merke: firstName ist der Parameter. "Max" und "Lisa" sind die Argumente (die konkreten Werte).

3. Parameter ≠ Variablenname

Der Name des Parameters und der Name der Variable, die du übergibst, müssen nicht gleich sein. JavaScript interessiert sich nur für den Wert. 

let myName = "Sara";

function sayHello(person) {
  console.log("Hi, " + person + "!");
}

sayHello(myName); // "Hi, Sara!"
myName "Sara" → person "Sara"

Die Variable myName gibt ihren Wert an den Parameter person weiter.

4. Lokale vs. Globale Variablen

Globale Variablen leben außerhalb von Funktionen und sind überall sichtbar.
Lokale Variablen (und Parameter!) existieren nur innerhalb ihrer Funktion. 

let color = "blau";          // global

function changeColor() {
  let color = "rot";          // lokal – neue Variable!
  console.log(color);         // "rot"
}

changeColor();
console.log(color);           // "blau" – global unverändert!
Global let color = "blau"
Lokal (Funktion) let color = "rot"
Achtung: Wenn du in einer Funktion let weglässt, überschreibst du die globale Variable! Nutze immer let oder const.

5. Rückgabewerte richtig nutzen

Mit return gibt eine Funktion einen Wert zurück. Diesen kannst du in einer Variable speichern.

Falsch:

function double(number) {
  let result = number * 2;
  // kein return! Der Wert geht verloren.
}

let result = double(5);
console.log(result); // undefined!

Richtig:

function double(number) {
  return number * 2;
}

let result = double(5);
console.log(result); // 10
Merke: Ohne return gibt jede Funktion undefined zurück. Der Rückgabewert muss in einer Variable gespeichert werden, sonst ist er weg!

6. Geführte Aufgabe: Der Witz-Generator

Jetzt wird es ernst! Folge den Schritten und baue Schritt für Schritt einen Witz-Generator in VSCode. Erstelle dafür eine neue Datei jokes.js und binde sie in eine HTML-Datei ein.

Vorbereitung: Erstelle eine uebung.html mit diesem Grundgerüst:

<script src="jokes.js"></script>

Öffne die Datei im Browser und öffne die Konsole (F12 → Console).
1

Das Witze-Array (globale Variable)

Erstelle ein Array mit mindestens 6 Witzen. Dieses Array ist eine globale Variable – es lebt außerhalb aller Funktionen und ist überall verfügbar.

Lösung anzeigen 
const jokes = [
  "Warum können Geister nicht lügen? Man kann durch sie hindurchsehen.",
  "Ich wollte einen Witz über Pizza machen... aber der ist zu cheesy.",
  "Warum trinken Programmierer Tee? Weil Java schon vergeben ist.",
  "Was macht ein Pirat am Computer? Er drückt die Enter-Taste.",
  "Treffen sich zwei Wände. Sagt die eine: Wir sehen uns an der Ecke!",
  "Warum hat der Mathematiker Angst vor negativen Zahlen? Er kann einfach nicht ohne sie leben."
];
Beachte: const statt let – das Array soll nicht neu zugewiesen werden. Die Inhalte können trotzdem geändert werden.
2

Zufälligen Index berechnen

Schreibe eine Funktion getRandomIndex, die eine Anzahl als Parameter bekommt und einen zufälligen Index zwischen 0 und (Anzahl - 1) zurückgibt.

Lösung anzeigen 
function getRandomIndex(count) {
  let index = Math.floor(Math.random() * count);
  return index;
}
jokes.length 6 → count 6 → return z.B. 3

Die Variable jokes.length wird zum Parameter count. Anderer Name, gleicher Wert!

3

Witz aus dem Array holen

Schreibe eine Funktion getJoke, die das Witze-Array als Parameter bekommt. Sie soll getRandomIndex() aufrufen und den Witz an der zufälligen Stelle zurückgeben.

Lösung anzeigen 
function getJoke(jokeList) {
  let index = getRandomIndex(jokeList.length);
  let selectedJoke = jokeList[index];
  return selectedJoke;
}
Wichtig: Hier ruft eine Funktion eine andere Funktion auf! Der Rückgabewert von getRandomIndex() wird in der lokalen Variable index gespeichert. Beachte auch: der Parameter heißt jokeList, nicht jokes!
4

Witz bewerten

Schreibe eine Funktion rateJoke, die einen einzelnen Witz-Text als Parameter bekommt und je nach Länge eine Bewertung zurückgibt. Wenn der Witz kürzer als 40 Zeichen lang ist soll "Kurz und knackig!" zurückgegeben werden. Wenn der Witz kürzer als 75 Zeichen lang ist soll "Perfekte Länge!" zurückgegeben werden. Ansonsten soll "Puh, das war ein langer Witz!" zurückgegeben werden

Lösung anzeigen 
function rateJoke(jokeText) {
  let jokeLength = jokeText.length;

  if (jokeLength < 40) {
    return "Kurz und knackig!";
  } else if (jokeLength < 75) {
    return "Perfekte Länge!";
  } else {
    return "Puh, das war ein langer Witz!";
  }
}
Beachte: Der Parameter heißt jokeText – nicht joke oder jokes. Der Name beschreibt, was die Funktion erwartet: einen einzelnen Text, kein ganzes Array.
5

Alles zusammenführen

Schreibe die Hauptfunktion showJoke, die alles verbindet: Witz holen, bewerten und in der Konsole ausgeben. Sie nutzt die Rückgabewerte der anderen Funktionen!

Lösung anzeigen 
function showJoke(jokeList) {
  let joke = getJoke(jokeList);
  let rating = rateJoke(joke);

  console.log("=== Witz des Tages ===");
  console.log(joke);
  console.log("Bewertung: " + rating);
  console.log("Zeichenanzahl: " + joke.length);

  return joke;
}

Funktionsaufruf-Kette:

showJoke(jokes)
getJoke(jokeList)
getRandomIndex(jokeList.length)
rateJoke(joke)
6

Aufrufen und testen!

Rufe die Funktion am Ende deiner Datei auf. Speichere den Rückgabewert in einer Variable – so kannst du den Witz weiterverwenden. Gib den Witz auf der Konsole aus.

Lösung anzeigen 
// Einmal aufrufen
let myJoke = showJoke(jokes);

// Den Rückgabewert weiterverwenden
console.log("Gespeicherter Witz: " + myJoke);
Tipp: Lade die Seite mehrmals neu (F5) – jedes Mal kommt ein anderer Witz!
*

Bonus: Witz-Vergleich

Schreibe eine Funktion compareJokes, die zwei Witze holt, beide bewertet und den kürzeren als "Gewinner" ausgibt.

Lösung anzeigen 
function compareJokes(jokeList) {
  let joke1 = getJoke(jokeList);
  let joke2 = getJoke(jokeList);

  let rating1 = rateJoke(joke1);
  let rating2 = rateJoke(joke2);

  console.log("--- Witz-Duell ---");
  console.log("Witz 1: " + joke1);
  console.log("  → " + rating1);
  console.log("Witz 2: " + joke2);
  console.log("  → " + rating2);

  if (joke1.length < joke2.length) {
    console.log("Gewinner: Witz 1 (kürzer!)");
    return joke1;
  } else {
    console.log("Gewinner: Witz 2 (kürzer!)");
    return joke2;
  }
}

compareJokes(jokes);
