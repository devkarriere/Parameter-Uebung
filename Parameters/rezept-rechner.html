Projekt: Rezept-Rechner

Portionen umrechnen mit Funktionen & Parametern

Der Rezept-Rechner

Baue einen Rechner, der Rezept-Zutaten automatisch auf eine gewünschte Personenanzahl umrechnet. Du arbeitest mit Arrays, Funktionen, Parametern und Rückgabewerten.

Erstelle eine neue Datei rezept.js und binde sie in eine HTML-Datei ein.

1

Rezept-Daten als globale Variable

Erstelle ein Rezept-Objekt mit einem Namen, der Basis-Personenanzahl und einem Array von Zutaten. Jede Zutat hat eine Menge, eine Einheit und einen Namen.

Lösung anzeigen 
const recipe = {
  name: "Pfannkuchen",
  servings: 4,
  ingredients: [
    { amount: 300, unit: "g", name: "Mehl" },
    { amount: 3, unit: "Stk", name: "Eier" },
    { amount: 500, unit: "ml", name: "Milch" },
    { amount: 50, unit: "g", name: "Zucker" },
    { amount: 1, unit: "Prise", name: "Salz" },
    { amount: 30, unit: "g", name: "Butter" }
  ]
};
Beachte: Das Rezept ist für 4 Personen. Alle Mengen beziehen sich darauf.
2

Umrechnungsfaktor berechnen

Schreibe eine Funktion calculateFactor, die die Basis-Portionen und die gewünschten Portionen als Parameter bekommt und den Umrechnungsfaktor zurückgibt. Wenn wir also ein Rezept für 4 Personen haben und wir daraus 6 Portionen machen, dann übergeben wir 4 und 6 und erhalten den Faktor 1.5

Lösung anzeigen 
function calculateFactor(baseServings, targetServings) {
  let factor = targetServings / baseServings;
  return factor;
}
baseServings 4 targetServings 6 → return 1.5

6 Portionen ÷ 4 Basis-Portionen = Faktor 1.5

3

Einzelne Zutat umrechnen

Schreibe eine Funktion scaleIngredient, die eine einzelne Zutat und einen Faktor als Parameter bekommt. Sie soll ein neues Objekt mit der umgerechneten Menge zurückgeben. Runde das Ergebnis auf eine Nachkommastelle.

Die übergebene Zutat soll also Objekt übergeben werden (siehe dein Format aus Aufgabe 1). Die darin enthaltene Menge wird mit dem Faktor multipliziert, gerundet und wieder im gleichen Format als Objekt zurückgegeben

Tipp: Math.round() rundet auf ganze Zahlen. Um auf eine Nachkommastelle zu runden, multipliziere vorher mit 10, runde, und teile wieder durch 10:
Math.round(2.76 * 10) / 10 → 2.8
2.76 * 10 = 27.6 → Math.round(27.6) = 28 → 28 / 10 = 2.8
Lösung anzeigen 
function scaleIngredient(ingredient, factor) {
  let newAmount = ingredient.amount * factor;
  let rounded = Math.round(newAmount * 10) / 10;

  return {
    amount: rounded,
    unit: ingredient.unit,
    name: ingredient.name
  };
}
Wichtig: Der Parameter heißt ingredient (Singular) – die Funktion bearbeitet immer nur eine Zutat, nicht das ganze Array.
4

Alle Zutaten umrechnen

Schreibe eine Funktion scaleRecipe, die das Zutaten-Array, die Basis-Portionen und die Ziel-Portionen als Parameter bekommt. Sie soll calculateFactor und scaleIngredient aufrufen und ein Array mit allen umgerechneten Zutaten zurückgeben. Bedeutet, wenn die Ziel-Portion 6 und die Basis-Portionen des Rezepts 4 sind, wird die Funktion scaleIngredient mit den passenden Parametern aufgerufen

Lösung anzeigen 
function scaleRecipe(ingredients, baseServings, targetServings) {
  let factor = calculateFactor(baseServings, targetServings);
  let scaled = [];

  for (let i = 0; i < ingredients.length; i++) {
    let newIngredient = scaleIngredient(ingredients[i], factor);
    scaled.push(newIngredient);
  }

  return scaled;
}

Funktionsaufruf-Kette:

scaleRecipe(ingredients, 4, 6)
calculateFactor(4, 6) → 1.5
scaleIngredient(ingredients[0], 1.5)
scaleIngredient(ingredients[1], 1.5)
... (für jede Zutat)
5

Rezept formatiert ausgeben

Schreibe eine Funktion formatRecipe, die den Rezept-Namen, die Portionen und das skalierte Zutaten-Array als Parameter bekommt. Sie soll das Rezept formatiert in der Konsole ausgeben.

Lösung anzeigen 
function formatRecipe(recipeName, servings, ingredients) {
  console.log("=== " + recipeName + " ===");
  console.log("Portionen: " + servings);
  console.log("");

  for (let i = 0; i < ingredients.length; i++) {
    let ing = ingredients[i];
    console.log(ing.amount + " " + ing.unit + " " + ing.name);
  }
}
6

Alles zusammenführen und testen

Rufe die Funktionen auf: Rechne das Rezept auf 6 Portionen um und gib es in der Konsole aus.

Lösung anzeigen 
// === Rezept-Daten ===
const recipe = {
  name: "Pfannkuchen",
  servings: 4,
  ingredients: [
    { amount: 300, unit: "g", name: "Mehl" },
    { amount: 3, unit: "Stk", name: "Eier" },
    { amount: 500, unit: "ml", name: "Milch" },
    { amount: 50, unit: "g", name: "Zucker" },
    { amount: 1, unit: "Prise", name: "Salz" },
    { amount: 30, unit: "g", name: "Butter" }
  ]
};

// === Funktionen ===
function calculateFactor(baseServings, targetServings) {
  let factor = targetServings / baseServings;
  return factor;
}

function scaleIngredient(ingredient, factor) {
  let newAmount = ingredient.amount * factor;
  let rounded = Math.round(newAmount * 10) / 10;

  return {
    amount: rounded,
    unit: ingredient.unit,
    name: ingredient.name
  };
}

function scaleRecipe(ingredients, baseServings, targetServings) {
  let factor = calculateFactor(baseServings, targetServings);
  let scaled = [];

  for (let i = 0; i < ingredients.length; i++) {
    let newIngredient = scaleIngredient(ingredients[i], factor);
    scaled.push(newIngredient);
  }

  return scaled;
}

function formatRecipe(recipeName, servings, ingredients) {
  console.log("=== " + recipeName + " ===");
  console.log("Portionen: " + servings);
  console.log("");

  for (let i = 0; i < ingredients.length; i++) {
    let ing = ingredients[i];
    console.log(ing.amount + " " + ing.unit + " " + ing.name);
  }
}

// === Aufrufen und testen ===
let scaledIngredients = scaleRecipe(recipe.ingredients, recipe.servings, 6);
formatRecipe(recipe.name, 6, scaledIngredients);
Tipp: Probiere verschiedene Portionszahlen aus – z.B. 2, 10 oder 1!

Bonus: Gesamtkosten berechnen

Erweitere das Rezept um Preise pro Einheit und schreibe eine Funktion calculateCost, die die Gesamtkosten für die skalierten Zutaten berechnet.

Lösung anzeigen 
const recipeWithPrices = {
  name: "Pfannkuchen",
  servings: 4,
  ingredients: [
    { amount: 300, unit: "g", name: "Mehl", pricePerUnit: 0.002 },
    { amount: 3, unit: "Stk", name: "Eier", pricePerUnit: 0.35 },
    { amount: 500, unit: "ml", name: "Milch", pricePerUnit: 0.002 },
    { amount: 50, unit: "g", name: "Zucker", pricePerUnit: 0.002 },
    { amount: 1, unit: "Prise", name: "Salz", pricePerUnit: 0.01 },
    { amount: 30, unit: "g", name: "Butter", pricePerUnit: 0.01 }
  ]
};

function calculateCost(ingredients) {
  let total = 0;

  for (let i = 0; i < ingredients.length; i++) {
    let cost = ingredients[i].amount * ingredients[i].pricePerUnit;
    total = total + cost;
  }

  let rounded = Math.round(total * 100) / 100;
  return rounded;
}

let cost = calculateCost(recipeWithPrices.ingredients);
console.log("Gesamtkosten: " + cost + " €");
