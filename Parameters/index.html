Parameter & Variablen in JavaScript

Verstehe den Unterschied – Schritt für Schritt

1. Was sind Variablen?

Eine Variable ist ein Behälter mit einem Namen, in dem du einen Wert speicherst. 

let name = "Anna";
let age = 25;

console.log(name); // "Anna"
console.log(age);  // 25
Merke: let erstellt eine Variable. Der Name links vom = ist frei wählbar.

2. Was sind Parameter?

Ein Parameter ist eine Platzhalter-Variable innerhalb einer Funktion. Er bekommt seinen Wert erst, wenn die Funktion aufgerufen wird. 

function greet(firstName) {
  console.log("Hallo, " + firstName + "!");
}

greet("Max");   // "Hallo, Max!"
greet("Lisa");  // "Hallo, Lisa!"
Merke: firstName ist der Parameter. "Max" und "Lisa" sind die Argumente (die konkreten Werte).

3. Parameter ≠ Variablenname

Der Name des Parameters und der Name der Variable, die du übergibst, müssen nicht gleich sein. JavaScript interessiert sich nur für den Wert. 

let myName = "Sara";

function sayHello(person) {
  console.log("Hi, " + person + "!");
}

sayHello(myName); // "Hi, Sara!"
myName "Sara" → person "Sara"

Die Variable myName gibt ihren Wert an den Parameter person weiter.

4. Lokale vs. Globale Variablen

Globale Variablen leben außerhalb von Funktionen und sind überall sichtbar.
Lokale Variablen (und Parameter!) existieren nur innerhalb ihrer Funktion. 

let color = "blau";          // global

function changeColor() {
  let color = "rot";          // lokal – neue Variable!
  console.log(color);         // "rot"
}

changeColor();
console.log(color);           // "blau" – global unverändert!
Global let color = "blau"
Lokal (Funktion) let color = "rot"
Achtung: Wenn du in einer Funktion let weglässt, überschreibst du die globale Variable! Nutze immer let oder const.

5. Rückgabewerte richtig nutzen

Mit return gibt eine Funktion einen Wert zurück. Diesen kannst du in einer Variable speichern.

Falsch:

function double(number) {
  let result = number * 2;
  // kein return! Der Wert geht verloren.
}

let result = double(5);
console.log(result); // undefined!

Richtig:

function double(number) {
  return number * 2;
}

let result = double(5);
console.log(result); // 10
Merke: Ohne return gibt jede Funktion undefined zurück. Der Rückgabewert muss in einer Variable gespeichert werden, sonst ist er weg!
