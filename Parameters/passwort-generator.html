Projekt: Passwort-Generator

Sichere Passwörter bauen mit Funktionen & Parametern

Der Passwort-Generator

Baue einen Generator, der zufällige Passwörter erstellt und ihre Stärke bewertet. Du kombinierst mehrere Funktionen, die sich gegenseitig aufrufen und Parameter sowie Rückgabewerte nutzen.

Erstelle eine neue Datei passwort.js und binde sie in eine HTML-Datei ein.

1

Zeichenvorräte als globale Variablen

Erstelle vier String-Variablen, die jeweils einen Vorrat an Zeichen enthalten: Kleinbuchstaben, Großbuchstaben, Ziffern und Sonderzeichen.

Lösung anzeigen 
const lowercase = "abcdefghijklmnopqrstuvwxyz";
const uppercase = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
const digits = "0123456789";
const symbols = "!@#$%&*?+-_";
Beachte: Strings kann man wie Arrays mit [index] ansprechen: "abc"[1] ergibt "b".
2

Zufälliges Zeichen aus einem String holen

Schreibe eine Funktion getRandomChar, die einen Zeichen-String als Parameter bekommt und ein zufälliges Zeichen daraus zurückgibt. Denke an die Funktione Math.floor und Math.random

Lösung anzeigen 
function getRandomChar(charSet) {
  let index = Math.floor(Math.random() * charSet.length);
  let char = charSet[index];
  return char;
}
lowercase "abcde..." → charSet "abcde..." → return "m"

Der globale String wird als Parameter übergeben. Die Funktion kennt nur charSet, nicht den Originalnamen.

3

Zeichenvorrat zusammenbauen

Schreibe eine Funktion buildCharPool, die vier Boolean-Parameter bekommt: useLower, useUpper, useDigits und useSymbols. Jeder Parameter bestimmt, ob die entsprechende Zeichengruppe im Passwort verwendet werden soll. Die Funktion soll einen String zurückgeben, der nur die Zeichen der aktivierten Gruppen enthält. Nutze dazu die Variablen aus Schritt 1 und füge sie mit + zusammen.

Lösung anzeigen 
function buildCharPool(useLower, useUpper, useDigits, useSymbols) {
  let pool = "";

  if (useLower) {
    pool = pool + lowercase;
  }
  if (useUpper) {
    pool = pool + uppercase;
  }
  if (useDigits) {
    pool = pool + digits;
  }
  if (useSymbols) {
    pool = pool + symbols;
  }

  return pool;
}
Beachte: Die Funktion hat 4 Parameter! Jeder steuert, ob eine Zeichengruppe enthalten sein soll. Mit + werden Strings zusammengefügt.
4

Passwort generieren

Schreibe eine Funktion generatePassword, die die gewünschte Länge und den Zeichenvorrat als Parameter bekommt. Dieser Zeichenvorrat ist die Rückgabe aus der Funktion von Aufgabe 3. Die neue Funktion soll getRandomChar in einer Schleife aufrufen und das fertige Passwort als String zurückgeben.

Lösung anzeigen 
function generatePassword(length, charPool) {
  let password = "";

  for (let i = 0; i < length; i++) {
    let char = getRandomChar(charPool);
    password = password + char;
  }

  return password;
}

Funktionsaufruf-Kette:

generatePassword(12, pool)
getRandomChar(charPool) → "k"
getRandomChar(charPool) → "A"
getRandomChar(charPool) → "7"
... (für jedes Zeichen)
5

Passwortstärke bewerten

Schreibe eine Funktion ratePassword, die ein Passwort und die Pool-Größe (Anzahl verschiedener möglicher Zeichen) als Parameter bekommt. Bewerte die Stärke nach der Formel: Länge × Pool-Größe. Unter 200 ist "Schwach", unter 500 "Mittel", darüber "Stark". Die Rückgabe soll sowohl den berechneten Score als auch die Bezeichnung "Schwach", "Mittel" und "Stark" zurückgeben (Hinweis: Objekt mit zwei Attributen).

Lösung anzeigen 
function ratePassword(password, poolSize) {
  let score = password.length * poolSize;

  if (score < 200) {
    return { label: "Schwach", score: score };
  } else if (score < 500) {
    return { label: "Mittel", score: score };
  } else {
    return { label: "Stark", score: score };
  }
}
Beachte: Hier wird ein Objekt zurückgegeben – mit zwei Werten gleichzeitig. So kann der Aufrufer sowohl das Label als auch den Score verwenden.
6

Alles zusammenführen und testen

Schreibe eine Hauptfunktion createPassword, die alle vorherigen Funktionen verbindet: Pool zusammenbauen, Passwort generieren, Stärke bewerten und alles in der Konsole ausgeben.

Lösung anzeigen 
function createPassword(length, useLower, useUpper, useDigits, useSymbols) {
  let pool = buildCharPool(useLower, useUpper, useDigits, useSymbols);

  if (pool.length === 0) {
    console.log("Fehler: Mindestens eine Zeichengruppe auswählen!");
    return null;
  }

  let password = generatePassword(length, pool);
  let strength = ratePassword(password, pool.length);

  console.log("=== Neues Passwort ===");
  console.log("Passwort:  " + password);
  console.log("Länge:     " + password.length);
  console.log("Pool:      " + pool.length + " Zeichen");
  console.log("Score:     " + strength.score);
  console.log("Stärke:    " + strength.label);

  return { password: password, strength: strength, poolSize: pool.length };
}

// Testen!
createPassword(12, true, true, true, true);
createPassword(8, true, false, true, false);
createPassword(20, true, true, true, true);

Funktionsaufruf-Kette:

createPassword(12, true, true, true, true)
buildCharPool(true, true, true, true) → "abcABC012!@#..."
generatePassword(12, pool)
getRandomChar(pool) × 12
ratePassword(password, 72) → { label, score }
*

Bonus: Mehrere Passwörter vergleichen

Schreibe eine Funktion findStrongest, die eine Anzahl als Parameter bekommt, so viele Passwörter generiert und das stärkste zurückgibt.

Lösung anzeigen 
function findStrongest(count) {
  let bestPassword = null;
  let bestScore = 0;

  for (let i = 0; i < count; i++) {
    let result = createPassword(12, true, true, true, true);
    if (result.strength.score > bestScore) {
      bestScore = result.strength.score;
      bestPassword = result.password;
    }
  }

  console.log("--- Bestes von " + count + " Passwörtern ---");
  console.log(bestPassword);
  return bestPassword;
}

findStrongest(5);
